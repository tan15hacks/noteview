# NoteView

NoteView is a clean modern blog landing page concept for ideas, guides, and stories worth reading.

## Brand direction

Modern blog and digital journal inspired by Medium, Notion, and premium online magazines.

## Sections

- Header navigation
- Featured post hero
- Latest blog posts
- Browse by category
- Editor's picks
- Popular this week
- Newsletter signup
- About section
- Structured footer

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide Icons
- Vercel-ready deployment

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
